# Typing Speed Test

A simple browser-based typing speed test built with HTML, CSS, and JavaScript.

## Features

- Displays current WPM, accuracy, time, and error count
- Shows a progress bar while typing
- Offers a result overlay when the test completes
- Allows restarting with a new paragraph

## Files

- `index.html` — page structure
- `style.css` — styling and layout
- `script.js` — typing test behavior

## Usage

1. Open `index.html` in your browser.
2. Click in the typing area and start typing the displayed paragraph.
3. Review your WPM, accuracy, time, and error count.
4. Click "New Paragraph" to reset.
